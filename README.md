## 📖 Overview

This repository is a **focused collection of Java implementations and problems on Linear Data Structures**. Instead of covering everything superficially, this repo goes deep into **Stacks, Queues, Circular Queues, and Linked Lists**.

The goal is simple: **master the core structures that actually appear in coding interviews**.

---

## 📂 Repository Structure

* **`Stack/`** → Stack implementation and problems

  * Basic operations (push, pop, peek)
  * Expression evaluation (infix, postfix)
  * Next greater element, monotonic stack problems

* **`Queue/`** → Queue implementation and problems

  * Linear queue operations
  * Priority-based problems

* **`CircularQueue/`** → Circular queue implementation

  * Efficient memory usage using circular logic
  * Real-world queue handling problems

* **`LinkedList/`** → Linked list problems

  * Singly linked list
  * Doubly linked list
  * Reverse, cycle detection, merging lists

> Each folder contains structured Java programs with clear logic and problem-solving approaches.

## 🚦 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/YashvanthBalaji/java-dsa-practice.git
   ```

2. Navigate to any folder:

   ```bash
   cd Stack
   ```

3. Compile and run:

   ```bash
   javac FileName.java
   java FileName
   ```
